# interesting_photo
PHP ve MySQL kullanılarak geliştirilmiş bu fotoğrafçılık web sitesi, Anasayfa, Hakkımızda, Hizmetlerimiz, Galeri ve İletişim bölümlerini içerir. Galeri veritabanından dinamik olarak yüklenir, iletişim formu kullanıcı mesajlarını kaydeder. Modern, şık ve tamamen responsive tasarıma sahiptir.
